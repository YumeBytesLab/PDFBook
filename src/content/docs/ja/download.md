## どのファイルを選べばいい？

### Windows

- **セットアップインストーラー**（`...-windows-x64-setup.exe`）— 通常のインストーラー。
  スタートメニューとデスクトップにショートカットを作成します。
- **ポータブル版**（`...-windows-x64-portable.exe`）— インストール不要で起動。USBメモリや
  任意のフォルダに置いて使えます。
- Arm デバイス（例: Surface Pro X）では `x64` ではなく `arm64` 版を使用してください。

### macOS

- **Apple Silicon**（M1/M2/M3 以降）— `arm64` の `.dmg`。
- **Intel Mac** — `x64` の `.dmg`。

`.dmg` を開き、PDFBook を Applications フォルダにドラッグします。

### Linux

- **AppImage** — 実行権限を付けて起動。インストール不要。
- **deb** — Debian / Ubuntu 系: `sudo apt install ./PDFBook-*.deb`。
- ハードウェアに合わせて `x64` または `arm64` を選択してください。

## Microsoft Store

PDFBook は Windows の Microsoft Store からも入手できます。Store 版はアプリが自動で
更新されます。

## アップデート

新しいバージョンが公開されると、PDFBook は起動時に通知し、このページへ案内します。
Microsoft Store からインストールした場合は、Store 経由で更新されます。
