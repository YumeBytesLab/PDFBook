## Which file do I need?

### Windows

- **Setup installer** (`...-windows-x64-setup.exe`) — the normal installer; creates a
  Start-menu and desktop shortcut.
- **Portable** (`...-windows-x64-portable.exe`) — runs without installing; keep it on a
  USB stick or in any folder.
- On an Arm device (e.g. Surface Pro X), use the `arm64` build instead of `x64`.

### macOS

- **Apple Silicon** (M1/M2/M3 and newer) — use the `arm64` `.dmg`.
- **Intel Macs** — use the `x64` `.dmg`.

Open the `.dmg` and drag PDFBook to your Applications folder.

### Linux

- **AppImage** — make it executable and run it; no installation required.
- **deb** — for Debian, Ubuntu, and derivatives: `sudo apt install ./PDFBook-*.deb`.
- Use the `x64` or `arm64` file to match your hardware.

## Microsoft Store

PDFBook is also available through the Microsoft Store on Windows, which keeps the app
updated automatically.

## Updates

When a newer version is published, PDFBook lets you know on startup and links you back to
this page. Installs from the Microsoft Store update through the Store instead.
